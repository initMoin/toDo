# Free Feature Set Contract

ToDo's current personal productivity feature set is permanently treated as free. Future paid products may expand the app into projects, teams, shared workspaces, or business administration, but they must not remove or paywall the individual workflow that already exists.

## Free Personal Scope

- Individual ToDos and NanoDos
- Tags
- Basic and complex reminders
- Recurring reminders
- Sorting, grouping, and filtering
- Saved personal views when implemented for personal use
- Personal calendar and planning views when implemented for personal use
- This Device Only storage
- Sync with iCloud for Apple devices when available
- ToDo Sync for personal account sync when available

## Future Paid Expansion Boundary

Paid tiers should add operating modes beyond personal productivity:

- Projects
- Sections/lists inside projects
- Task dependencies
- Project templates
- Attachments and comments when project-scoped
- Shared workspaces
- Roles, assignments, owners, admins, and members
- Activity history and administrative controls
- Team notifications and business-oriented cross-platform collaboration

## Engineering Rule

Current personal features should not depend on future monetization checks. If a future capability system is added, personal capabilities must default to enabled and should be covered by tests.

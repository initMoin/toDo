# Security Policy

## Supported Versions

Currently, only the latest release of toDō on the App Store / TestFlight is supported with security updates. 

## Reporting a Vulnerability

Security is taken very seriously. If you discover a security vulnerability within toDō, please do not disclose it publicly via GitHub Issues. 

Instead, please report it directly by reaching out via **[security@iamshift.dev]**. 

All security vulnerabilities will be promptly investigated. Please provide as much detail as possible, including steps to reproduce the issue, so it can be verified and patched quickly.
